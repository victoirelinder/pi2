# mongodb-interface


### Installation

- open **mongod**
- run :
```
> use ENRON

> db.createCollection("emails")
```
- Clone my folder, go in it
- use  **Node.js** and run: 
```
> npm install
```
```
> npm start
```

- go to [localhost:3000](http://localhost:3000)
